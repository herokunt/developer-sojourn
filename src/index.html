About...

Hi! I'm a web developer based in Madrid, Spain.

I learned how to write code because I wanted to make sense of so many things about technology that fascinate me. The little marvels of the modern world that we give for granted every day: instant messaging, video streaming, 3D graphics, cryptography or "the cloud".

Many people predict that in the not-so-distant future coding will become an essential skill to have. I think learning how to code is so much more than that. It's an empowering, challenging and mind-opening exercise that requires patience and creativity. It pushes you to do better, learn something new, stay curious and sharp. That feeling of being able to make what I want however I want, is why I learned to code. A feeling there's not coming back from.

Beyond technology I enjoy the simple things the most: beer with friends, a good read, a hike in the mountains. I'm also an avid traveler and diver. If you want to know more about me, below you will find my GitHub page, email address and my PGP public key.

See what I'm up to and get in touch

Developer Sojourn is a place to relax, learn and experiment with new technologies. I attempt to put into practice everything that I learn about programming for the web, ranging from flashing visuals to the less obvious but equally important back-end setup.

I also write about my personal experience learning how to write code, mostly in the format of note taking blog posts, but I will try to keep the content mixed with some personal thoughts, news and other articles I find interesting. The main topics are about web development, game development and graphics, system administration, etc.

This portfolio is a work in progress, and it always will as I change things around, throw in new technologies to the stack, follow on design trends and find other overly complicated ways to have some fun with code.

Any feedback is much appreciated, I always welcome tips on how to improve my coding and designing skills. You can also just drop me a message to say hi or share your own work with me. Enjoy your visit!
