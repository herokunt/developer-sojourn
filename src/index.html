Web Apps...

Share files privately and securely, over the browser. Bytes And Pipes is a web application that allows to share large files very easily using only a web browser. Files are protected using end-to-end encryption during upload, which means its contents remain unknown to the server and potential eavesdroppers. Uploaded files expire in 24 hours or after they've been downloaded, and are automatically deleted afterwards.
